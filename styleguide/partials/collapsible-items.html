Warning:

Check your markup: If show-collapsed and hide-expanded classes are not behaving correctly it is possible that you added another class allong with these. For example: <span class="show-collapsed ma-all da-hide ta-hide">Show Stuff </span> will fail because ma-all overrides show-collapsed display settings. ma-all in this case should be a div parent to the show-collapsed.
