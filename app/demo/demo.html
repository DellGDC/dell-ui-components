Overview


Welcome to the Dell UI Documentation

This design library contains DCX-approved components in the Polaris/Simplified visual identity, using Bootstrap version 3.1.1.

The full UX documentation for these components can be found in the current library at delldesignlibrary.com.


Before you explore

Please take note of some of the features you will see through the pages on the right:


Copy HTML buttons

Once you copy your desired component you will see confirmation that the code in in your clipboard and can be pasted in your HTML editor.

Components with color treatments

Components that can inherit colors will have dropdowns with the color options. Select your desired color first and then when you copy the component code the color class will be in your clipboard.
