Styleguide rules:

NOTE: Dell button class names deviated greatly from Bootstrap. They are now normalized using standard Bootstrap class names.

  • .btn-default should be deprecated default .btn should be used instead.
  • .btn-info, .btn-warning, .btn-danger and .btn-inverse are not to be used per the design styles.
  • Use class no-arrow with a dropdown-toggle to suppress the dropdown arrow in a dropdown button or link but not a split button
