Navbar search

<div navbar-search="searchTerm" data-placeholder="What would you like to do?"></div>
With custom callback function to execute on submit.
<div navbar-search="searchTerm" data-submit="doSomethingWithSearchTerm" data-placeholder="What would you like to do?"></div>
About callback function: "doSomethingWithSearchTerm" refers to a function you would have in the controller for the page or component you have this form element in. Example: 

	        $scope.doSomethingWithSearchTerm = function(searchTerm) {
	          alert('hi you searched for "'+searchTerm+'"');
	        };
