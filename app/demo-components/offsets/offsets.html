Design guidance:

The deprecated offset classes like "-none -mini -small -medium -large" can still be used. However, we strongly encourage you to use the pixel modifier classes like "-0 -5 -10 -20 -30 -40 -60".

For example, simply place class="top-offset-10" into a element if you want the element to have a top offset margin of 10px.

Dev guidance:

You should use offset classes as sparingly as possible. Often a div is lacking a vertical space because it does not have the inside content properly formatted inside p tags. Please note that p tags and .well already have a bottom margins.
