Icons Demo

Dev guidance: The following icons are specific to Dell Brand Standards. Please use the following icon class names in place of native Bootstrap icon class names.

Color guidance: All of the icons below can change to the following colors by adding these class modifiers .text-blue, .text-dark-blue, .text-purple, .text-berry, .text-red, .text-red-dark, .text-gray-medium, .text-gray-dark, .text-orange, .text-green, or .text-white. For example: class="icon-ui-arrowleft text-blue" change the modifier color after the .text-*.

Small icons

The small icons have less detail and are intended to be used at the default font size of 16px. Please use the small icons for task based communications. For crisper small icons use the following sizes: 16px, 28px and 42px.
