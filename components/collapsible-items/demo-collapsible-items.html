Create a system restore point for your computer

System Restore is a Microsoft® Windows® tool designed to take a "snapshot" of select system files and the Windows registry and save them as restore points. When an install failure or data corruption occurs, System Restore can return a system to working condition by reverting to the files and settings that were saved in the last restore point.
