Dell-UI-Components Development Environment

The purpose of this guide is to provide guidance on working on the Dell-UI-Components package. If you have any suggestion for things that should be clarified or improved place contact Bo Lora.

The information in this guide is extremely important. If you will be contributing code or patches to dell-ui-components, you will need to strictly follow these development guidelines. Code which doesn't follow the guidelines will degrade the dell-ui-components package and make it harder for maintenance in the future.

Some code already in place may not match these guidelines, this is mostly due to it being written before the guidelines were defined. Assistance with cleaning up any incorrect code is very inportant to the future of the package but it should be properly coordinated through our process. If you see anything that needs to be fixed, please contact Sunny Boukadakis.
